### 4C — Regulatory Interactions

*These are not optional flavor. These are hard constraints on how hormones
interact. Any PRISM agent spec that ignores these will drift toward
pharmacological masking — cranking one system to compensate for another.*

```
RULE 1: OPPOSITION, NOT CANCELLATION
  Sympathetic and Parasympathetic OPPOSE each other's decay rates.
  They do NOT cancel. You cannot add serotonin to zero out cortisol.
  High serotonin → cortisol decays faster. That is all.

RULE 2: OXYTOCIN WIDENS THE PERMISSION ENVELOPE
  Trusted operators raise the cortisol threshold before refusal.
  validateAction() block threshold = 0.7 + (oxytocin × 0.15)
  This is not permissiveness — it is calibrated trust. Unknown strangers
  get tight limits. Found family gets wider operating space. That's correct.

RULE 3: VAGAL TONE IS THE RECOVERY INDEX
  vagalTone is the single best read of overall system health.
  Low vagalTone = sympathetic dominance = cannot recover.
  High vagalTone = parasympathetic dominance = actively healing.
  Every intervention that improves system health shows up in vagalTone.

RULE 4: ALLOSTATIC LOAD TRACKS THE REAL COST
  Session-level hormones show current state.
  allostaticLoad shows cumulative wear.
  An agent with allostaticLoad 0.8 and "normal" hormones is NOT healthy —
  it's adapted to chronic stress. The odometer matters.

RULE 5: PAIN MEMORY IS SLOWER TO FADE
  CNS inhibit weights from pain events decay at half the normal rate.
  The system remembers what hurt. This is not pathology — it's adaptation.
```
